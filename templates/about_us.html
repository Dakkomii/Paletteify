    Tools Utilized

    Color Thief
    Created by Lokesh Dhakar, Color Thief is an API that extracts dominant colors from an image input.
    Learn More
    Webcolors
    A Python library. It allows the conversion from certain RGB values to their CSS color name equivalents.
    Learn More
    JSColor
    A JavaScript library. It's a color picker dropdown that we utilized to allow customization to profile page backgrounds.
    Learn More
    RGB Color Classifier
    Ajinkya Chavan worked with Tensorflow to create a way of color classifying that helped us with categorizing posts into our 10 aesthetic categories.
